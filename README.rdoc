ucp-auth

Login => http://your_url/sessions (POST)
params email, password
return false or current_user

is_signed =>  http://your_url/sessions/signed_in (GET)
return true or false if user is login

logout =>  http://your_url/sessions/destroy (GET)
destroy user session

profile => http://your_url/users/profile (GET)
get current_user 

user_formation => http://your_url/users/users_formation (GET)
get all students from current_user promo
